# VPN Extras

Static support assets for hwdsl2 VPN projects.

This repository includes helper scripts, registry files, and release assets used by VPN setup scripts and Docker images. Some Docker VPN images use public GitHub release asset download counts for anonymous, aggregate usage counts. These counts help understand usage and prioritize future improvements. The projects do not send a telemetry payload or use a private collector.

Current summaries can be found in the [stats](stats) folder.
